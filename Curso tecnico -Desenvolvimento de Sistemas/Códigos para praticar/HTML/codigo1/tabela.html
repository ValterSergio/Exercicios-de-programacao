Tabelas em HTML

Tabelas podem ser criadas com a tag <table>, toda tabela possui colunas e linhas, por questão de inteligencia é sempre nomear as colunas, que ao mesmo tempo compõe a primeira linha da tabela, essa primeira linha da tabela pode ser chamada de cabeçalho de tabela, e como foi percebido, precisamos de uma LINHA para fazer esse cabeçalho então usamos a tag <tr> dentro da tabela para criar a primeira linha, e usamos a tag <th> para essa finalidade.
  Para adicionar dados precisamos da tag <td> e toda linha que for criada na tabela deve começar com <tr>.
